# riyanshi.dev — portfolio

An interactive desktop-OS style portfolio for **Riyanshi Kedia** — data scientist & AI engineer.
Built with plain HTML, CSS, and JavaScript (no build step). Inspired by the PostHog aesthetic,
with a reckon.house-style skill constellation.

## Features
- Draggable, closable, minimizable app windows (Home, Projects, Education, Experience, Contact)
- `skills.viz` — an animated canvas skill constellation
- Light / dark theme toggle
- Fully responsive

## Run locally
```bash
python3 -m http.server 4321
# then open http://localhost:4321
```

No dependencies — it's just static files.
